# Mini Calculator Game

A simple and interactive web-based calculator that performs basic arithmetic operations with a beautiful user interface.

## Features

### Basic Operations
- **ADD** - Addition of two numbers
- **SUB** - Subtraction of two numbers
- **DIV** - Division of two numbers (with zero-division error handling)
- **MULT** - Multiplication of two numbers
- **MOD** - Modulo (remainder after division)
- **POW** - Power (raise number to an exponent)
- **SQRT** - Square root (requires only one number)
- **PERC** - Percentage calculation

### Other
- **EXIT** - Exit the calculator

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (ES6)

## How to Run

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start calculating!

## Usage

1. Select an operation by clicking one of the buttons (ADD, SUB, DIV, MULT, MOD, POW, SQRT, PERC)
2. Enter the first number (and second number if required)
3. Click "Calculate" to see the result
4. Click "Back to Menu" to perform another calculation
5. Click "EXIT" to close the calculator
